### 2. 函数级 — 八股段落（按等级输出）

**S 级**（跨模块接口 / 状态机 / 资源申请释放）：
- 完整八股：破题 + 承题 + [起讲] + 入手 + 起股 + 中股 + 后股 + 束股

**A 级**（核心业务算法 / 数据流转换）：
- 微八股：破题 + 起股 + 中股 + 后股 + 束股

**B 级**（工具函数 / 纯计算无状态）：
- 极简八股：破题 + 起股 + 中股 + 束股

**C 级**（常量 / UI 布局 / 配置）：
- 单行破题或裸写

### 3. 对偶标记

- `✓` / `✗` — 正路径 vs 降级路径
- `🔒` / `🔓` — 资源申请 vs 释放（必须成对）
- `↗` / `↘` — 计数器增减、标志位切换

### 4. 空八股规则

允许空八股，但格式统一为 `# <股名>：N/A` 或 `# <股名>：无需`，禁止省略整股。

### 5. 格式铁律

- 所有注释行必须以 `# ` 开头（# 后有空格）
- 股标记统一为 `==== X股 ====`（8个= + 空格 + 股名 + 空格 + 8个=）
- 破题必须严格两句，分号分隔，第二句以"不做"开头
