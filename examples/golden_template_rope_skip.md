## 关键参数（已调优，不建议改）

| 参数 | 值 | 说明 |
|------|-----|------|
| TAKEOFF_TH | 0.475 | 起跳阈值，norm_y 低于此值判定起跳 |
| LANDING_TH | 0.50 | 落地阈值，norm_y 高于此值判定落地 |
| MIN_AIR_TIME | 0.03s | 最小腾空时间，防抖动 |
| MIN_INTERVAL | 0.23s | 最小计数间隔，防双计 |
| MAX_AIR_TIME | 2.0s | 最大腾空时间，超时强制落地 |
| 滑动窗口 | 10s / 250帧 | 减少基线漂移 |
| 置信度阈值(肩) | 0.3 | 肩膀可见度门槛 |
| 置信度阈值(髋) | 0.4 | 髋部可见度门槛（fallback 更严） |

## 适配其他动作

| 动作 | 破题修改 | 信号源修改 | 状态机修改 |
|------|---------|-----------|-----------|
| 跑步计时 | 检测腿部交替周期 | 髋/膝/踝 | 左右脚相位 |
| 立定跳远 | 检测起跳→腾空→落地全程 | 髋中心 | 单次检测，不循环 |
| 仰卧起坐 | 检测肩-髋角度变化 | 肩-髋-膝三点 | 积分制状态机 |
| 俯卧撑 | 检测肩-肘-腕角度 | 肩-肘-腕三点 | 低位→高位翻转 |

**结构不变：起股(取/验) → 中股(算/转) → 后股(正/降) → 束股(给/留)**
